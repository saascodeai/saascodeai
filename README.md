## What Ships In Every Product

### 🤖 Full MCP — 87+ Tools

A complete Model Context Protocol server across 19 categories. Payments, users, plans, features, integrations, automations, email, theme — all controllable through natural language. Your AI agent discovers what the product can do, configures it, and runs it.

### 💳 18 Global Payment Providers

Stripe · PayPal · Paddle · Mollie · MercadoPago · Razorpay · dLocal · PagSeguro · Paystack · Flutterwave · Coinbase Commerce · NOWPayments · Lemon Squeezy · Alipay · WeChat Pay · Komoju · Opn/Omise · KakaoPay

Americas, Europe, Africa, Asia, crypto. Universal webhook translator. Customers pay however they want, wherever they are.

### 🔐 28 Auth Providers

17 native + 10 proxy + Snapchat. Identity linking, progressive disclosure login, admin management API sync.

### 🔌 100+ Integrations

Pre-configured. 15 categories. 5 connection patterns. Analytics, marketing, support, monitoring, CRM — toggle on from the admin panel.

### ⚡ Three-Layer Automation System

Event automations (reactive) → Scheduled tasks (natural language cron) → Report templates (metric DSL). Chain them: a task runs a report and sends it via email, Slack, or webhook. Zero code.

### 🌍 i18n From Line One

Every string translatable. 15 languages in the setup wizard. No hardcoded strings, no "we'll add i18n later."

### 🎨 Page Composer

19 section types, 54+ variants, 12 hero layouts. Visual editor with wireframe previews. Build landing pages without code.
